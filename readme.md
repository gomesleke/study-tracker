# 📚 Study Tracker (Controle de Estudos)

## 📌 Sobre o Projeto

O **Study Tracker** é uma aplicação em Python desenvolvida para registrar e controlar o tempo dedicado aos estudos por disciplina.

O sistema funciona via terminal (CLI) e permite iniciar sessões de estudo com cronômetro, registrar a duração da sessão e salvar automaticamente os dados em arquivos JSON.

O objetivo do projeto é praticar organização de código, modularização, manipulação de arquivos e estruturação de dados.

---

## 🚀 Funcionalidades

* ⏱️ Cronômetro de estudo em tempo real
* 📂 Criação automática de arquivos JSON
* 🗂️ Separação de sessões por disciplina
* 📊 Registro estruturado de horas, minutos e segundos

---

## 🏗️ Estrutura do Projeto

* `manager.py` → Responsável pela criação, leitura e salvamento de arquivos JSON.
* `time_study.py` → Controla o cronômetro e registra as sessões.
* `time_subject` → Gerenciamento das disciplinas.
* Arquivos `.json` → Armazenamento das sessões de estudo.
